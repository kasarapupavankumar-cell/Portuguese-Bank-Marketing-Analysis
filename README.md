# Portuguese Bank Marketing Analysis

This project focuses on predicting whether a customer will subscribe to a term deposit using machine learning techniques.

## Objective
To help the bank identify potential customers and improve marketing campaign efficiency.

## Dataset
Portuguese Bank Marketing Dataset containing customer demographic, campaign, and economic information.

## Steps Performed
- Data cleaning and duplicate removal
- Exploratory Data Analysis (EDA)
- Feature encoding and scaling
- Handling class imbalance using SMOTE
- Model training and evaluation
- Model comparison and final model selection

## Models Used
- Logistic Regression
- Logistic Regression with SMOTE
- Random Forest
- Gradient Boosting
- XGBoost

## Final Model
XGBoost was selected as the final model due to its highest recall, ensuring maximum identification of potential subscribers.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

## Business Insight
Recall was prioritized over accuracy, as missing potential customers is costlier than contacting additional customers.
